== Description ==
This plugin will add gallery meta-fields for the post area. You can create galleries mixed with video and images.

You have to use embeded video links. Tested with You-Tube.  

== Usage ==
Shortcode `[bsImageGallery id='1']`

== Requirements ==
Recommanded PHP Version: 5.6+
